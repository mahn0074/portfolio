Projekt:
Stilart site

Final aflevering af grundlæggende webdesign er en responsive stilartsite.
Og der er mange opgaver i løbet af denne proces for at lære hvordan man kan lave en hjemmeside. I starten skulle stilarten analyseres, visualiseres og samles materialer i form af Moodboard, Stilart Sites, og Content til stilart site.
Vi skal lære Html og Css og lave øvelser, såsom: visitkort, flexbox og kæledyr øvelser o.s.v. Disse øvelser hjælper os til at laver vores stilart site:
For at programmere Stilartsite begynder man at lave en layout diagram i html filen, som består af feks sectionwraper, container, comlun-narrow o.s.v.
Splash billederne og ikonerne bliver designet i Photoshop. Font, Ikon, Splash sættes ind i html og css til final project aflevering. For at lave en responsive hjemmeside lærer vi om hvordan man skriver kode som media queries.
